# Журнал изменений

Все заметные изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.2] - 2026-03-17

### Исправлено

- Улучшена обработка путей и каталогов в задаче покрытия.
- Уточнена обработка значений в функции получения свойства объекта.

### Изменено

- Упрощена логика и улучшена читаемость обработки значений в модулях `PrometheusTextFormat` и `PrometheusVec`.

### Документация

- Добавлена отдельная пользовательская документация для библиотеки prometheus.
- Обновлен README: исправлена ссылка на DeepWiki, добавлена информация о реализации HTTP-сервиса для отдачи метрик и новые значки/ссылки.

### Прочее

- Обновлены правила кодирования и очищены комментарии в файлах задач.
- Обновлен `.gitignore` и добавлен служебный файл `fake-entrypoint.os`.

## [1.0.0] - 2026-03-08

### Добавлено

- Первый релиз библиотеки сбора метрик Prometheus для OneScript.
- Реестр коллекторов (CollectorRegistry), типы метрик: Counter, Gauge, Histogram, Summary.
- Vec-варианты с лейблами (СчетчикВектор, ИндикаторВектор, ГистограммаВектор, РезюмеВектор).
- Сериализация в Prometheus Text Format; фасад модуль Prometheus для единой точки входа.
- Низкоуровневые модули: PrometheusMetrics, PrometheusVec, PrometheusRegistry, PrometheusTextFormat.
